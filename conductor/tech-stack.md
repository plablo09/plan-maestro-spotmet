# Technology Stack

## Backend
- **Python (FastAPI):** A fast and modern web framework for building APIs.
- **DuckDB with Spatial Extension:** Used for efficient storage and retrieval of spatial data in vector tile formats (MVT).

## Data Processing
- **GeoPandas:** For calculating accurate building areas and handling spatial data transformations.
- **Fiona, Pyogrio, Pyarrow:** Supporting libraries for efficient data reading and storage (GeoPackage, Parquet).

## Frontend
- **HTML/CSS:** Standard web technologies for structure and styling.
- **MapLibre GL JS:** An open-source map visualization library used for the high-performance 3D building rendering and interactions.
